# Maze Solver - BFS on Python


A simple maze solver using BFS algorithm with GUI on Python. Made for the course of Artificial Intelligence at State University of Surabaya. 

---
## How to run
```bash
poetry install
```
```bash
poetry shell
```
```bash
python .\bfs_maze_solver\MazeSolver.py 
```
